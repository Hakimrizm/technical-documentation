What is PHP

PHP is a general-purpose scripting language geared toward web development. It was originally created by Danish-Canadian programmer Rasmus Lerddorf in 1994. The PHP reference implementation is now produced by The PHP Group.

What php can do?

  • PHP can generate dynamic page content.
  • PHP can create, open, read, write, delete, and close files on the server.
  • PHP can collect form data.
  • PHP can send and receive cookies.
  • PHP can add, delete, modify data in your database.
  • PHP can be used to control user-access.
  • PHP can encrypt data.
Hello World

A "Hello, World!" program is generally a computer program that outputs or displays the message "Hello, World!". This program is very simple to write in many programming language, and is often used to illustrate a language's basic syntax.

Example:

echo "Hello, World!";
Variables

Variables in program are used to store some values or data that can be used later in a program. The variables are also like containers that store character values, numeric values, memory addresses, and strings. PHP has it's own way to declaring and storing variables.

In PHP variable starts with the $ sign, followed by the name of the variable.

$txt = "Hello world"; $x = 5; $y = 10.5l
If else

If statement executes some code if one condition is true. if...else statement executes some code if a condition is true and another code if that condition is false.

The if statement executes some code if one condition is true.

PHP - The if Statement.

if (condition) { code to be executed if condition is true; }

PHP - The if...else Statement.

if (condition) { code to be executed if condition is true; } else { code to be executed if condition is false; }
While Loop

In most computer programming languages, a while loop is a control flow statement that allows code to be executed repeatedly based on a given Boolean condition. The while loop can be thought of a as a repeating if statement.

The example below displays the numbers from 1 to 5:

$x = 1; while($x <= 5) { echo "The number is : $x"; }
